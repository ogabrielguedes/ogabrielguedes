# 👨‍💻 Gabriel Guedes
**`Desenvolvedor Front-End`**

Em transição do Marketing para o código, com paixão por criar experiências web intuitivas e performáticas. Minha bagagem em estratégias digitais me ensinou a importância de design centrado no usuário, interfaces limpas e otimização de performance — agora aplico isso no desenvolvimento.

### 🚀 **Foco Atual:**

Fundamentos Front End: HTML, CSS (Flexbox, Grid), JavaScript (ES6+);

React.js: Desenvolvendo interfaces dinâmicas com hooks e componentes;

Design Responsivo & Acessibilidade: Garantindo experiências fluidas em qualquer dispositivo;

Ferramentas: Git, VS Code, Figma (para colaboração em UI).

### 💡 **Por que Desenvolvimento?**
Acredito que produtos incríveis nascem onde lógica e criatividade se encontram. Como dev, busco escrever código limpo e reutilizável, sempre com o usuário final em mente.

### 🔍 **Buscando:**
Oportunidades para colaborar em projetos, contribuir para open-source e evoluir na prática.
